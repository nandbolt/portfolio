ILIAGC: I Live In A Graphing Calculator

  • Overview: A graphing calculator + 2D platformer with a focus on education.
  • Made with: GameMaker, GML
  • Solo project
  • Development: Done
  • Origin: Started as a class project.
  • Technical challenges:
    • Graphing calculator: There were many hurdles that needed to be overcome for the graphing calculator. The player had to be able to type in any valid math function anywhere in the level. The equation needed to be parsed into something that could be evaluated so that it could be graphed. Lastly, the graph should be interactable by the player in a physically believeable way.
