Planar Escape

  • Overview: Push blocks around to solve puzzles and escape while being chased by a dangerous entity.
  • Made with: GameMaker, GML
  • Solo project
  • Development: Ongoing
  • Origin: A remake of my first game with the intention of testing out the physics engine I made.
  • Technical challenges:
    • Physics: Rather than using GameMaker's built-in physics engine, I created a particle-based, iterative, impulse-based physics engine within GameMaker using the guidance of Ian Millington's book Game Physics: Engine Development which is written in C++.
    • Level editor: Allowed for players to create custom levels, save them locally and easily share level files with other players.
